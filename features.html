Canvas

Alternative Token Radial Menu

Replaces the default token radial token menu with a more compact and simplistic one.

Animation Disable

Disables certain animations.

Transparent Canvas UI Dialogs

Provides a way to adjust opacity on canvas UI dialogs.

Display Current Layer on Canvas

Displays the currently active editing layer on the canvas in a more obvious way using bright colors and text. No more "why can't I select this token".

Middle Click Switch To Token Layer

Allows middle clicking on a token to switch to it's layer.

Draw Token Layer on Tokens

Very obviously displays which tokens are on which layer so you know exactly which token a layer is on.

Initiative

Automatically Sort Initiative

Automatically sorts initiative when new tokens are added without resetting the order.

Ping Token

Pings a token on it's turn if it's visible to the players.

Select Token

Selects a token on it's turn.

Move Camera to Token

Moves the local camera to a token on it's turn.

Advance Turn Shortcut

Binds Ctrl+Left Arrow key to advance the initiative.

Journal

Duplicate button

Adds a duplicate button to the context menu of items in the journal.

Macro Generator

Generates attack, spell, feat etc macros for characters.

Token

Bulk Macros

Allows rolling a single macro for a group of tokens. Great for initiative and group checks. Use via the token context menu.

Roll & Apply Hit Dice

Allows rolling of hit dice and automatically applies it. Use via the token context menu.

Reassign "Represents" when duplicating

Updates which character represents a token when duplicating that character.

Exporting/Importing

Characters

Export and import ANY character (PC, NPC) to JSON.

Rollable Tables

TableExport format importing is supported!

Macros

Transfer your macro collection across games.
